# mynote2.0
这是一个android便笺类应用。 

**实现的功能：**

1. 初次使用注册用户名。 

2. 以后使用可以直接到登录界面。登录可以选择记住密码和自动登陆两种选择。 

3. 点击底部的+可以添加便笺，默认点击返回键会自动保存。 

4. 点击便笺列表可以进入便笺编辑界面，对便笺进行编辑，默认点击返回键会自动保存。 

5. 长按便笺列表的某一项，可以唤出上下文菜单，对便笺进行删除。 

6. 便笺列表是按照最后修改时间排序的。  

**亮点：**

1. 应用activity的生命周期对应用程序的内存使用进行了优化。 

2. 界面比较美观，小米风~~

**说明**

1. 使用IDEA开发。

2. 要求Android 2.1版本以上。

3. 版本2.0使用sqlite实现。
